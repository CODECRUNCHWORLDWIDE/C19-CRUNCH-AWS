# Week 1 — Challenges

The exercises drill the basics. **Challenges stretch you.** This one takes 90–150 minutes and produces something real you can put in your portfolio: a billing-observability pipeline that proves you can answer "what is this costing, by service and by tag?" before you have launched a single server.

## Index

1. **[Challenge 1 — Cost & Usage Report → S3 → Athena](challenge-01-cur-to-athena.md)** — deliver the CUR to an S3 bucket, register it in Athena with partition projection, and write SQL that breaks spend down by service and by tag. (~120 min)

Challenges are optional for passing the week. If you do this one, you will be measurably ahead — and the CUR → Athena pipeline you build here is exactly the one Week 14 (FinOps) extends into a QuickSight dashboard, and the one every later week's cost report queries. Build it well now and you reuse it for fourteen weeks.
